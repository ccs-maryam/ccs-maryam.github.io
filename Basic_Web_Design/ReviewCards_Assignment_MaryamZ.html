Basic Web Design - Transformation Flashcards

What does “scale3d(x,y,z);” do?

It defines a 3D scale transformation.

What does “scaleX(n/x);” do?

It defines a 2D or 3D scale transformation; it changes the element's width.

What does “transform:” do?

It applies a 2D or 3D transformation to an element.

What does "rotateX(angle);" do?

It defines a 3D rotation along the x-axis.

What does “transform-origin:” do?

It changes the position of transformed elements.

What does “perspective:” do?

It specifies the perspective on how 3D elements are viewed.

What does “skewX(angle);” do?

It defines a 2D skew transformation along the x-axis.

What does “translate3d(x,y,z);” do?

It defines a 3D translation.

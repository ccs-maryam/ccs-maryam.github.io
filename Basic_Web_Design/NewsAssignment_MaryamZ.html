Mega-Makers Class Offered at the Whittier Central Library

The Whittier Central Library has recently begun to offer a Mega-Makers class, a monthly class that focuses on teaching teens and adults how to design basic 3D objects in a free online program called Tinkercad. The class is held in a specially designated room called the Maker Space which can be found on the second floor of the library. Those who attend the class can request to have a design they make to be printed by one of the 3D printers the library has, and an additional request can be made per each class that a person attends.

Community Impact

The Mega-Makers class is one of the many ways the Whittier Central Library is striving to engage with the community. Like other classes the library has started to offer, such as its Intro to Sewing Machines class, the Mega-Makers class allows Whittier residents to engage in an activity alongside other members of their community, all while exploring an interesting field of design.
